WHAT IS CLOUD COMPUTING

Cloud computing is the on-demand availability of computer system resources, especially data storage and computing power, without direct active management by the user. Large clouds often have functions distributed over multiple locations, each location being a data center.

What is cloud computing examples? Image result for cloud computing Examples would include: Dropbox, a file storage and sharing system. Microsoft Azure, which offers backup and disaster recovery services, hosting, and more. Rackspace, which offers data, security, and infrastructure services.
